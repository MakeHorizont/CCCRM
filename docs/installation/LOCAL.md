
# Инструкция: Локальная Разработка ("Верстак")

Товарищ разработчик! Эта инструкция поможет тебе развернуть среду разработки на локальной машине.

## 1. Подготовка Инструментов
Убедись, что в твоем арсенале есть:
*   **Node.js** (версия 18 или выше).
*   **npm** или **yarn**.
*   **Git**.

## 2. Получение Кода
Клонируй репозиторий:
```bash
git clone https://github.com/MakeHorizont/CCCRM.git
cd CCCRM
```

## 3. Настройка Окружения
Система требует API ключ для работы AI-функций (Gemini).

1.  Создай файл `.env` в корне проекта (или `.env.local`):
    ```bash
    cp .env.example .env
    ```
2.  Впиши свой ключ:
    ```env
    GEMINI_API_KEY=твой_ключ_здесь
    ```
    *Примечание: Ключ можно получить в Google AI Studio. Без него система будет работать, но AI-ассистент и авто-генерация отзывов будут недоступны.*

## 4. Установка Зависимостей
Установи необходимые пакеты. Мы используем строгие версии для дисциплины.
```bash
npm install
```

## 5. Запуск
Запусти сервер разработки:
```bash
npm run dev
```
Открой браузер по адресу: `http://localhost:3000`

## 6. Режим работы с Бэкендом
На данном этапе (версия 1.8.0) система работает в гибридном режиме.
*   В файле `services/api/config.ts` можно переключать флаг `USE_REAL_API`.
*   По умолчанию (`false`) система использует мощные моки (mockData), которые эмулируют работу сервера. Данные сбрасываются при перезагрузке страницы.
*   Если вы разрабатываете Бэкенд (NestJS), установите флаг в `true` и следуйте инструкциям в `documents/backend/`.
