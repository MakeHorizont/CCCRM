## Версия 1.7.1 (24.11.2025) - "Аналитический Центр"

### Новые Функции
-   **Модуль "Аналитика":**
    -   Создан централизованный раздел для визуализации ключевых метрик предприятия.
    -   **Вкладка "Производство":** Графики динамики выпуска продукции (План vs Факт) за последние 30 дней. Таблица эффективности по продуктам.
    -   **Вкладка "Экономика":** График динамики себестоимости единицы продукции. Позволяет отслеживать реальную стоимость производства (материалы + труд) по завершенным партиям.

### Технические Улучшения
-   **Библиотека Графиков:** Внедрены легковесные CSS/SVG компоненты (`LineChart`, `BarChart`) для отображения данных без внешних зависимостей.
-   **Сервис Аналитики:** Реализована логика агрегации исторических данных из производственных и складских модулей.

## Версия 1.7.0 (24.11.2025) - "Мобильный Цех"

### Новые Функции
-   **Терминал Цеха (Kiosk Mode) 2.0:**
    -   **Живая Экономика:** Внедрен блок "Сводка Смены" на главном экране терминала.
    -   **Таймер:** Отображает время, прошедшее с начала смены.
    -   **Заработок:** Показывает расчетную сумму заработка за сегодня в реальном времени (Оклад + Сделка).
    -   **КТУ:** Динамическая индикация текущего Коэффициента Трудового Участия.
    -   **Визуализация Режимов:** Адаптивный интерфейс терминала меняет цветовую схему на красную при включении режима "Мобилизация", повышая внимание персонала.

### Технические Улучшения
-   **User Service:** Добавлен метод `getDailyStats` для расчета оперативных показателей сотрудника без полной выгрузки истории.

## Версия 1.6.2 (24.11.2025) - "Цифровой Госплан"

### Новые Функции
-   **Материальная Диалектика (Physics Engine):**
    -   Внедрена жесткая логика для режима "Мобилизация". Создание или запуск производственных заданий теперь **физически блокируется** системой при дефиците сырья. Ошибку нельзя обойти, только пополнить склад.
-   **Аналитический Центр:**
    -   Добавлен виджет **"Производственный Пульс"** на Дашборде.
    -   Визуализация выполнения плана в реальном времени (План/Факт).
    -   Расчет эффективности производства (%).
    -   Отображение количества отстающих заказов.

## Версия 1.6.1 (24.11.2025) - "Экономика Справедливости"

### Новые Функции
-   **Прозрачная Формула Зарплаты:**
    -   В профиле сотрудника (вкладка "Финансы") теперь визуализируется алгоритм начисления: `База + (Сделка × КТУ) + Премии`.
    -   Выделено влияние КТУ на итоговую сумму, чтобы мотивировать качество работы.
-   **Улучшения Терминала Цеха:**
    -   Добавлена кнопка **"Закончить смену"** для точного учета рабочего времени.
    -   Добавлена кнопка **"Сообщить о проблеме"** (Инцидент) для быстрой регистрации брака, поломок или нехватки сырья прямо с планшета.
-   **Системные События:**
    -   События начала и конца смены теперь фиксируются в Журнале Событий.

## Версия 1.6.0 (24.11.2025) - "Кибернетический Коммунизм"

### Новые Функции
-   **Режим "Терминал Цеха" (Kiosk Mode):**
    -   Запущен специализированный упрощенный интерфейс для планшетов в производственной зоне.
    -   Крупные кнопки, темная тема для снижения энергопотребления и нагрузки на глаза, авторизация без лишних шагов.
    -   Функционал: Старт смены, Взятие ПЗ в работу одной кнопкой, Трекинг выполнения.
-   **Системные Режимы ("Материальная Диалектика"):**
    -   Внедрен глобальный переключатель режимов работы предприятия:
        -   **"Развитие":** Стандартный режим, допускает гибкость и ручные корректировки.
        -   **"Мобилизация":** Кризисный режим. Жесткая блокировка производственных действий при дефиците сырья (Physics Engine). Требует строгого соблюдения регламентов.
-   **Экономика Рационализации:**
    -   Интеграция модуля "Обсуждения" с финансами.
    -   Механизм внедрения рацпредложений с фиксацией экономического эффекта.
    -   Автоматическое начисление премии автору (10% от экономии) через системную транзакцию.
-   **Механизм "Двойного Ключа" (Совет):**
    -   Изменение критических параметров (например, % отчислений в Фонд) теперь невозможно единолично.
    -   Система требует создания Предложения в Совет и набора кворума голосов для автоматического применения изменений.

### Улучшения
-   **Аудит (Стеклянный Завод):** Расширено логгирование критических действий (изменение профилей, финансы, производство).
-   **Интерфейс:** Добавлены визуальные индикаторы режима мобилизации и дефицита сырья.
